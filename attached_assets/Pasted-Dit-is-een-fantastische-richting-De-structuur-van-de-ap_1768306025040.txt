Dit is een fantastische richting. De structuur van de app van je vriend (de "Uitklapbare Menu's") is perfect om heel veel functies (die je nu mist) overzichtelijk te maken, zonder dat het scherm te druk wordt.

We gaan die structuur combineren met de "Midnight Grand Slam" (Nike/Pro Tennis) look.

Hier is de volledig nieuwe PlayerHomeScreen.

Wat is er nieuw?
De Header (Player Card): Geïnspireerd op de gele balk van je vriend, maar dan in "Pro Tech" stijl. Donkere achtergrond met felle Electric Green accenten voor je naam, rank en status.

Het "Smart Menu" (Accordion): Dit is de kern. We hebben 4 hoofdcategorieën gemaakt waar ALLE functies in zitten die je miste.

MY CAREER: Hier zitten je Stats, Swing Lab (Video), Rankings en Drills.

ACTION CENTER: Boeken, Matchen en Toernooien.

ACADEMY HUB: Shop, Nieuws en je Squad (Ouders).

SYSTEM: Wallet en Instellingen.

De "Pro" Look: Diepblauwe achtergronden, strakke witte tekst en felle neon-groene iconen als je een menu opent.

// --- COMPONENT: HET UITKLAPBARE MENU ITEM ---
const ExpandableMenuItem = ({ item, isExpanded, onToggle }) => {
    const Icon = item.icon;
    const activeColor = isExpanded ? PRO_TENNIS_COLORS.electricGreen : PRO_TENNIS_COLORS.white;
    const bgColor = isExpanded ? PRO_TENNIS_COLORS.surfaceLight : 'transparent';

    return (
        <View style={[styles.menuItemContainer, { backgroundColor: bgColor }]}>
            {/* De Hoofdknop */}
            <TouchableOpacity 
                style={styles.menuHeader} 
                onPress={onToggle}
                activeOpacity={0.7}
            >
                <View style={styles.menuHeaderLeft}>
                    <Icon size={24} color={activeColor} style={styles.menuIcon} />
                    <Text style={[styles.menuTitle, { color: activeColor }]}>{item.title}</Text>
                </View>
                {isExpanded ? (
                    <ChevronUp size={20} color={activeColor} />
                ) : (
                    <ChevronDown size={20} color={PRO_TENNIS_COLORS.muted} />
                )}
            </TouchableOpacity>

            {/* De Sub-items (Alleen zichtbaar als expanded) */}
            {isExpanded && (
                <View style={styles.subMenuContainer}>
                    {item.subItems.map((subItem) => {
                        const SubIcon = subItem.icon;
                        return (
                            <TouchableOpacity 
                                key={subItem.id} 
                                style={styles.subMenuItem}
                                onPress={() => {
                                    Haptics.selectionAsync();
                                    subItem.action();
                                }}
                            >
                                <SubIcon size={18} color={PRO_TENNIS_COLORS.muted} style={styles.subIcon} />
                                <Text style={styles.subMenuText}>{subItem.title}</Text>
                            </TouchableOpacity>
                        )
                    })}
                </View>
            )}
        </View>
    );
}

// --- HOOFD SCHERM COMPONENT ---
const PlayerHomeScreenNew = () => {
  const [expandedSection, setExpandedSection] = useState<string | null>('career'); // Career staat standaard open

  const toggleSection = (id: string) => {
    // Trigger de animatie
    LayoutAnimation.configureNext(LayoutAnimation.Presets.easeInEaseOut);
    Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Light);
    setExpandedSection(expandedSection === id ? null : id);
  };

  return (
    <SafeAreaView style={styles.safeArea}>
      <StatusBar barStyle="light-content" />
      <View style={styles.container}>
        {/* Achtergrond met subtiel patroon (optioneel) */}
        <LinearGradient
            colors={[PRO_TENNIS_COLORS.midnightBlue, '#000000']}
            style={StyleSheet.absoluteFill}
        />

        <ScrollView contentContainerStyle={styles.scrollContent}>
          {/* 1. De Header (Player Card) */}
          <PlayerHeader />

          {/* 2. Het Uitklapbare Menu */}
          <View style={styles.menuContainer}>
            {MENU_STRUCTURE.map((item) => (
                <ExpandableMenuItem
                    key={item.id}
                    item={item}
                    isExpanded={expandedSection === item.id}
                    onToggle={() => toggleSection(item.id)}
                />
            ))}
          </View>
        </ScrollView>
      </View>
    </SafeAreaView>
  );
};

Wat zie je nu?
Header: Een strakke, donkere header met jouw naam, een "Semi-Pro" rank en een "Form" balk die felgroen is. Rechtsboven zie je je volgende actie.

Menu: Een lijst met 4 dikke knoppen: "MY CAREER", "ACTION CENTER", etc.

De Actie: Tik op "MY CAREER". Het klapt soepel open (met een kleine trilling/haptic feedback). De titel wordt felgroen.

De Inhoud: Daaronder verschijnen direct de functies die je zocht: Stats, Swing Lab, Rankings, Drills. Alles is nu 1 tik weg.

Dit is de perfecte mix tussen de structuur van je vriend's app en de "Pro Tennis" look die we wilden.
