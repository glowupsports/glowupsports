🧱 Maak dit BULLETPROOF (belangrijk!)
1️⃣ Centraliseer API calls (best practice)

Maak bv:

// api.ts
export const apiFetch = (path: string, options?: RequestInit) => {
  return fetch(getApiUrl(path), {
    credentials: "include",
    ...options,
  });
};


En gebruik ALLEEN dit:

apiFetch("/api/coaches/directory");


👉 Dit voorkomt dat dit ooit weer gebeurt
👉 Dit is enterprise-grade hygiene
