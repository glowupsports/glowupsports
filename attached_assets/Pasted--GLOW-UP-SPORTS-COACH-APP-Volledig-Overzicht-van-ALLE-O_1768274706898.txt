🎾 GLOW UP SPORTS – COACH APP
Volledig Overzicht van ALLE Openstaande Issues, Gaten & Risico’s

Dit is alles wat nog niet af is, half af is, fout kan gaan of een launch/beta in gevaar brengt.

🔴 A. KRITIEKE PROBLEMEN (BETA / LAUNCH BLOCKERS)

Deze dingen moeten opgelost zijn voordat je dit serieus met coaches laat werken.

1. ❌ Real-time Chat (WebSocket)

Status: UI bestaat, maar functioneel onbruikbaar
Impact: Coaches kunnen niet fatsoenlijk communiceren

Problemen:

Geen WebSocket verbinding

Geen live updates

Geen:

Typing indicators

Read receipts

Push bij nieuw bericht

Messages verversen alleen bij reload

Niet gekoppelde API’s:

/api/coach/messages/realtime

/api/coach/messages/send

Betrokken screens:

ChatInboxScreen

CoachChatFooter

👉 Zonder dit is coach–player communicatie kapot

2. ❌ Push Notifications (Triggers ontbreken)

Status: Token registratie oké, triggers ontbreken
Impact: Coaches missen cruciale momenten

Wat niet gebeurt:

Geen push bij:

Nieuwe sessie

Sessie wijziging

Nieuwe feedback request

Chatbericht

Session starting soon

Alleen visuele notificaties in app

👉 Voor coaches is dit zeer frustrerend

3. ❌ Video Playback (Skill Evidence)

Status: Opnemen werkt, terugkijken niet
Impact: Coach kan werk niet evalueren

Problemen:

Geen video player

Evidence kan niet:

Teruggekeken worden

Vergeleken worden

Met parent besproken worden

Niet gekoppelde API:

/api/coach/evidence/:id/video

4. ❌ Offline Mode (Coaches op baan = geen internet)

Status: Niet aanwezig
Impact: App kan tijdens training onbruikbaar zijn

Problemen:

Geen lokale cache

Geen offline feedback

Geen queue voor sync later

👉 Dit is serieus belangrijk voor coaches op de baan

🟠 B. GROTE FUNCTIONELE GATEN (HIGH PRIORITY)
5. ⚠️ Chat = Geen Real-Time + Geen Betrouwbaarheid

Extra problemen bovenop WebSocket:

Geen retry bij failure

Geen “message failed” status

Geen last-seen indicator

6. ⚠️ Push Notifications Niet Configureerbaar

Probleem:

Coach kan niet instellen:

Welke notificaties wel/niet

Stilte periodes

Prioriteiten

7. ⚠️ My Reviews = Read-only

Status: Alleen bekijken

Wat ontbreekt:

Reageren op review

Context toevoegen

Dispute / feedback uitleg

👉 Coaches willen reageren, zeker bij lage reviews.

8. ⚠️ Coach Invitations Workflow Onzeker

Problemen:

Accept/decline flow onduidelijk

Geen duidelijke status feedback

Geen reminder voor open invites

🟡 C. UX / PRODUCT PROBLEMEN (MID PRIORITY)
9. ⚠️ Nieuwe Coach = Overweldigend

Probleem:

Te veel tegelijk

Geen “wat nu?” guidance

Ontbreekt:

Guided onboarding na login

Contextual tips

Focus mode voor beginners

10. ⚠️ Geen Empty States

Probleem:

Lege schermen zonder uitleg

Betroffen o.a.:

Players (nieuwe coach)

Calendar (geen sessies)

Chat inbox

Reviews

11. ⚠️ Geen Skeleton Loaders

Probleem:

Abrupte loading

Slechte perceived performance

12. ⚠️ Complexe Wizards (Cognitive Load)

Probleem:

CreateSessionWizard = 6 stappen

Veel keuzes tegelijk

Kan beginners afschrikken

👉 Niet fout, maar gevaarlijk zonder guidance

🟡 D. TECHNISCHE RISICO’S (MID PRIORITY)
13. ⚠️ Extreem Grote Files

Probleem: Onderhoudbaarheid & bugs

Top offenders:

CoachingScreen – 5.248 regels

CalendarScreen – 4.104 regels

DashboardScreen – 3.356 regels

SeriesDetailDrawer – 3.408 regels

👉 Kleine wijziging = groot risico

14. ⚠️ Geen Error Boundaries

Probleem:

Eén runtime crash = hele coach app weg

Geen fallback UI

15. ⚠️ Query Over-Fetching

Probleem:

Dashboard doet veel overlappende calls

Calendar refetcht te vaak

Impact:

Performance issues

Onnodig dataverbruik

16. ⚠️ Geen Calendar Export

Probleem:

Coaches willen Google Calendar / iCal

Bestaat niet

API bestaat maar niet gebruikt:

/api/coach/calendar/export

🎯 CONCLUSIE (KEIHARD EERLIJK)

De Coach App is:

Visueel ~95% af

Functioneel ~85% af

Operationeel sterk

Communicatie & infrastructuur = zwakke plek

Het is een geweldige coach tool,
maar zonder chat, push en offline fallback gaan coaches klagen.
