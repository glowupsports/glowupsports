🟦 TECHNIQUE (ELITE-STABIEL)

Techniek is niet meer het focuspunt, maar het fundament.

FOREHAND

⬜ FH = duidelijke identiteit (spin / tempo / druk)

⬜ Kan tempo dicteren, niet alleen volgen

⬜ Inside-out / inside-in automatisch

⬜ Kan winners forceren zonder over-risico

⬜ Verliest FH niet onder druk

BACKHAND

⬜ BH is geen target voor tegenstanders

⬜ Kan rally’s controleren via BH

⬜ DTL BH betrouwbaar bij druk

⬜ Kan tempo vertragen of versnellen

SERVE

⬜ Serve is een wapen

⬜ Placement + disguise

⬜ Tweede service nog steeds agressief

⬜ Serve +1 is default patroon

RETURN

⬜ Neemt initiatief op return

⬜ Leest serve tendencies

⬜ Kan return winner / directe druk creëren

NET / FINISH

⬜ Afmaken is efficiënt, niet flashy

⬜ Weet exact wanneer net te kiezen

⬜ Geen “panic plays”

🟨 TACTICAL (DOMINANT DENKEN)

Glow 4 spelers spelen de tegenstander, niet de bal.

⬜ Analyseert opponent vóór de match

⬜ Past tactiek aan per set / game

⬜ Herkent momentum shifts vroeg

⬜ Speelt score extreem volwassen

⬜ Kan winnen zonder “beste spel”

🔥 Glow 4 = schaakspel

🟩 PHYSICAL (ELITE DUURZAAM)
