#126 - Play Screen — Find Players deep-link + filter verbeteringen
What & Why
Three small but impactful UX fixes in the Play screen and homescreen navigation.

Done looks like
Pressing "Find Players" anywhere on the homescreen opens Play and lands directly on the Players tab (not Group Lessons)
The Players tab opens with "Same Level" as the default selected filter, showing players at the user's own ball level first
All four filter chips (All, Recommended, Same Level, Open to Play) fit on screen at once without scrolling
Fixes
Fix 1: "Find Players" on homescreen → Players tab
File: client/player/components/SessionHeroCard.tsx (or wherever the "Find Players" text link under "No Sessions Today" is rendered — check the COURT TIME card on PlayerHomeScreen) Problem: The "Find Players" link navigates to PlayStack but likely does NOT pass { screen: "Play", params: { initialTab: "Players" } } — so it lands on Group Lessons. Fix: Find all "Find Players" navigation calls in the homescreen and make sure they all pass initialTab: "Players":

navigateToTab("PlayStack", { screen: "Play", params: { initialTab: "Players" } });
Check these files:

client/player/screens/PlayerHomeScreen.tsx
client/player/components/SessionHeroCard.tsx
Any other component on the homescreen with a "Find Players" button
Fix 2: Default Players filter = "Same Level"
File: client/player/screens/PlayScreen.tsx line ~133 Problem: useState<DiscoverFilter>("all") — starts on All players. Fix: Change to useState<DiscoverFilter>("sameLevel") so the player immediately sees others at their own level.

Fix 3: Filter chips smaller — all visible at once
File: client/player/screens/PlayScreen.tsx — styles.discoverChip Problem: paddingHorizontal: Spacing.md and paddingVertical: Spacing.sm make chips too wide to all fit on screen. Fix: Reduce chip padding so all 4 chips fit. Suggested:

paddingHorizontal: Spacing.sm (reduce horizontal)
paddingVertical: 4 (tighter vertical)
Icon size: 12 (from 14)
Text: slightly smaller than Typography.small, e.g. fontSize: 11
Remove text labels for icons where icon is self-explanatory, OR keep short labels
Target: all 4 chips visible without horizontal scroll
