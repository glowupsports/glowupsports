🎴 1️⃣ COMMUNITY FEED CARD — DE BASIS

Dit is het hart van je tennis social world.
Elke card moet in 0,5 seconde duidelijk maken:

Wie postte dit? Wat gebeurde er? Waarom is dit cool?

🧱 Card structuur (van boven naar beneden)
┌────────────────────────────┐
│ 👤 Avatar  Naam   🎖 Title │
│     Academy / Group        │
├────────────────────────────┤
│                            │
│   📸 FOTO / 🎥 VIDEO       │
│   (60–70% van card)        │
│                            │
│  🏷 Training | Match | etc │
├────────────────────────────┤
│ ✨ Caption (max 1 zin)     │
├────────────────────────────┤
│ 👏  💬  ➕  XP+5           │
│ 23 cheers   4 reacties     │
└────────────────────────────┘

🔥 Waarom dit werkt

Foto eerst → emotie

Context label → geen chaos

Acties onderaan → dopamine

XP zichtbaar → game loop

🎨 VISUELE STIJL (belangrijk)

Donkere achtergrond (premium / gaming)

Subtiele glow rond:

avatar

XP badge

moment-type icon

Ronde hoeken (vriendelijk voor kids)

Grote touch-targets (mobile-first)

🏷 MOMENT TYPE BADGES (heel belangrijk)

Elke post heeft exact 1 type, met eigen kleur & icon:

🎾 Training → groen

🏆 Match → goud

👥 Group → blauw

🎉 Event → paars

🏅 Achievement → roze

🌱 Free Play → cyaan

➡️ Zo kan het brein scannen, niet lezen.

In group posten	+bonus

➡️ XP animatie licht op bij actie
➡️ Kleine sparkle / pulse → dopamine
