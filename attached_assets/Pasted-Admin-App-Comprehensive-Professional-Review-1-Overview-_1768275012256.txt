Admin App - Comprehensive Professional Review
1. Overview of the Webapp
Purpose
The Admin App is a multi-academy management console for Academy Owners and Platform Owners. It provides a centralized dashboard to manage all aspects of a tennis academy including coaches, players, sessions, payments, courts, and reporting.

Architecture
Frontend: React Native + Expo (client/admin/)
Navigation: Bottom Tab Navigator (7 tabs) + Stack Navigator (5 additional screens)
State Management: TanStack Query for server state
API: RESTful endpoints with role-based access control (admin, academy_owner, platform_owner)
Styling: Dark gaming theme with orange accent color (Colors.dark.orange)
