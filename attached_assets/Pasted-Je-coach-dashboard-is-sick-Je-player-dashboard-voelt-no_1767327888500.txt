Je coach dashboard is sick.
Je player dashboard voelt nog te veel als software.

Dat verschil mag er zijn, maar niet zó groot.

Laten we dit scherp trekken.

🧠 Het fundamentele verschil (waar het misgaat)
🧑‍🏫 Coach dashboard = Command Center

Actief

Controlerend

Actie-gedreven

Tijd + beslissingen

“Ik run iets”

➡️ Dat heb je PERFECT neergezet.
Session module, live timer, actions, earnings — top.

🎮 Player dashboard = Game HUD

Niet:

administratie

overzicht

instellingen

Maar:

“Wat is mijn volgende quest?”

En daar gaat het nu nét mis.

❌ Waarom je player dashboard nog geen game voelt

Zelfs met de verbeteringen voelt het nog:

te netjes

te logisch

te informatief

Dat komt door dashboard-patronen:

Cards met titels

Statische blokken

Informatieve copy

Alles is “leesbaar” maar niet “speelbaar”

Games doen dit anders.

✅ Hoe een GAME dashboard voelt (belangrijk)

Een game dashboard heeft:

ÉÉN focuspunt (nu / live / next)

Status bars (niet cijfers)

Actieknoppen met consequentie

Veranderende UI op basis van state

Geen uitleg — alleen hints

🔥 Hoe je PLAYER dashboard een GAME maakt (concreet)

Ik ga dit 1-op-1 spiegelen aan je coach dashboard, want dat is je benchmark.

🥇 1. Next Training = PLAYER QUEST

Stop met denken:

“Next Training card”

Denk:

🎯 ACTIVE QUEST

Wat je nu toont:

Next training

Cancel

Running late

Wat het moet zijn:

Quest titel

“Group Training – Maple”

Countdown timer (groot!)

Status chip

“UPCOMING”, “STARTING SOON”, “LIVE”

Acties voelen als keuzes, niet knoppen

🟥 Cancel
🟨 Running Late
🟦 Message Coach

➡️ Dit is letterlijk je player equivalent van de coach session module.

🥈 2. Player Status = HUD (niet cards)

Je streak / XP / energy moeten voelen als:

🎮 Health / XP / Momentum

Niet:

“0 days”

“5000 XP”

Maar:

🔥 Momentum: LOW / BUILDING / HOT

⚡ Energy bar

🧠 Focus indicator

➡️ Bars + icons
➡️ Zo min mogelijk tekst

🥉 3. Academy moet verdwijnen naar de achtergrond

In een game:

Je zit niet elke seconde in het “menu”

De world is impliciet

Dus:

❌ Grote academy cards
❌ “Member since”
❌ Organisatie-info

✅ Subtiele context:

Training at: Glow Up Tennis Academy

Meer niet.

🟢 4. Coach = NPC / Mentor

De coach card is goed, maar denk zo:

“Dit is mijn mentor in de game”

Toon:

Coach avatar

Naam

Badge (Head Coach / Pro / Elite)

CTA: View Coach

Geen uitleg, geen bio op home.

🧠 5. Dynamische states (DIT maakt het een game)

Dit is waar jij écht kunt winnen.

Player dashboard verandert op basis van:
Situatie	Wat speler ziet
Training binnen 60 min	Countdown + urgency
Training live	BIG LIVE CARD
Geen training	“No active quests”
Vakantie	Paused state
Te laat	Warning state
Net training gehad	Reward animation

👉 Dashboards veranderen niet. Games wel.

🧪 Concrete vergelijking (jij vs TennisLocker)
TennisLocker player:

Schema

Lessen

Admin

Ouderlijke vibe

Jouw player (als je dit doet):

Live countdown

Actions

Feedback loop

Progress feeling

“Ik ben bezig met iets”

👉 Dat is waarom jouw coach dashboard al zo goed voelt.

“Ik ga vooruit”
