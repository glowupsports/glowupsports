Concept: jouw rating = “Glow MMR” + “Glow Rank buckets”

Glow MMR is een getal bv. 0–3000

Glow Rank is de bucket:

0–300 = Glow 9

301–600 = Glow 8

…

(tuning later, maar basis staat)

Skill gates blokkeren promotie naar hogere bucket.

Match update (Elo-achtig maar beter)

Je wilt 3 dingen in je rating update:

opponent strength

score margin

match verification trust

Stap 1 — Expected outcome
expected = 1 / (1 + 10^((oppMMR - myMMR)/400))

Stap 2 — Outcome
outcome = 1 (win) or 0 (loss)

Stap 3 — Margin factor

Gebruik set/game verschil om “close win” minder hard te maken dan “dominant win”.

marginFactor = clamp(0.85 + (gamesDiff * 0.03), 0.85, 1.25)

Stap 4 — Trust factor (anti-cheat)
trustFactor =
  1.00 if system-verified (in-app scoring both confirm)
  0.85 if coach-verified
  0.70 if self-reported (only one side)

Stap 5 — K-factor (activity/stability dependent)

Meer matches = stabieler = lagere K.

K = baseK * activityFactor * volatilityFactor
baseK = 28
activityFactor = 1.0 if >=6 matches/8w else 0.85
volatilityFactor = 1.15 if new player (<10 matches total) else 1.0

Final rating update
delta = K * (outcome - expected) * marginFactor * trustFactor
newMMR = myMMR + delta

Anti-farming rules (keihard)

Max 2 matches per 7 dagen tellen tegen dezelfde opponent voor rating

Wins tegen opponent >2 ranks lager tellen “reduced”

Self-reported match zonder confirm → telt 70% en kan niet promoten zonder skill gate bewijs

Rank promotion rules (met gates)

Een player kan alleen naar hogere Glow Rank bucket als:

MMR threshold gehaald

Min match volume gehaald (per 8 weken)

Skill gates ✅ (voor dat rank)

En voor hogere ranks:
4) Stability check (geen extreme swings / no rage quits)

Pseudo-code (super simpel)
function updateGlowRankAfterMatch(player, match) {
  if (isFarming(match)) return;

  const expected = expectedScore(player.mmr, match.opp.mmr);
  const outcome = match.didWin ? 1 : 0;

  const marginFactor = calcMargin(match.gamesDiff);
  const trustFactor = calcTrust(match.verification);
  const K = calcK(player.activity, player.matchesCount);

  const delta = K * (outcome - expected) * marginFactor * trustFactor;
  player.mmr += delta;

  const targetRank = mmrToRank(player.mmr);

  if (canPromote(player, targetRank)) {
    player.glowRank = targetRank;
    rewardLevelUp(player);
  }
}
