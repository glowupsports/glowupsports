A. LOOKS – hoe maak je van 7 → 10/10
1. Kleurgebruik: nu veilig, straks premium

Wat je nu hebt:

Donkergrijs / zwart

Neon groen (primary)

Soms cyaan / geel / rood

Probleem:
Alles is “mooi”, maar alles is even belangrijk. Daardoor mist het hiërarchie.

🔧 Oplossing (klein maar krachtig)

Gebruik 3 vaste niveaus van kleurintensiteit:

① Primary Glow (actie / focus)

Alleen voor:

CTA’s (Save, Book, Add)

Actieve tab

“Up next”

Iets meer glow + iets lichter groen

② Soft Status (informatie)

Energy, wellbeing, load

Minder verzadigd groen

Meer matte gradient (rust)

③ Accent (zeldzaam!)

Geel → waarschuwing

Rood → fout / absent

Blauw/cyaan → info

👉 Regel: als alles groen is, is niets groen.
Je zit hier nu net iets te royaal.

2. Cards: ze mogen meer “diepte”

Je cards zijn strak, maar nu iets te vlak.

Upgrade zonder redesign:

Iets sterkere:

inner shadow (boven)

zachte glow onderaan

Mini-gradient in cards (heel subtiel)

Denk:

“Apple + gaming”, niet “flat dashboard”

3. Typografie-hiërarchie (belangrijk!)

Je font is goed, maar:

Headers, labels en data zijn soms te gelijk

Cijfers (tijd, minuten, XP) mogen dominanter

Aanpassing:

Cijfers → iets groter + iets lichter

Labels → iets kleiner + meer opacity

Section titles → net iets meer spacing

Resultaat: rust + premium feeling.

4. Icon consistency

Je icons zijn goed, maar:

Niet allemaal even “gewicht”

Kies één icon-weight (bijv. outline of semi-filled)
en gebruik die overal.

Dit maakt ‘m ineens professioneler zonder dat je het kunt aanwijzen.

B. UX – kleine dingen die ‘m af maken
1. Empty states: al goed, maar kan beter

Je empty states zijn netjes, maar je mist 1 ding:

👉 “Wat moet ik nu doen?”

Bijv.:

“No completed lessons today”
→ + “Your next session starts in 39 min”

“No templates yet”
→ + “Most coaches create 3–5 templates”

Dat maakt de app coachend, niet alleen informerend.

2. Dashboard: iets meer persoonlijkheid

Je dashboard is sterk, maar kan nóg meer “Coach HQ” worden.

Kleine upgrade:

Toon:

“Today you coach 3 players”

“120 min on court”

Of: “Light day” / “Heavy day” als titel

Dat voelt als:

“Deze app kent mijn dag.”

3. Snellere acties (1-tap)

Overweeg later:

Long-press op session → Attendance

Swipe op player → Quick note

Swipe op day → Add session

Niet nu verplicht, maar dit is hoe je power users wint.

C. Is het klaar? (belangrijk antwoord)
Functioneel?

👉 Ja. Voor coach usage is dit klaar om live te gebruiken.

Je kunt:

Lessen plannen

Aanwezigheid doen

Feedback geven

Load beheren

Availability instellen

Templates gebruiken

Dit is meer dan genoeg om sheets te vervangen.

Maar… wat zou ik NOG aanpassen vóór publicatie?
MUST (klein):

Kleurhiërarchie aanscherpen (zie A)

Icon consistency

Typografie hiërarchie

NICE (later):

Player dashboard (light v1)

Push notifications

Real-time chat

Mijn eerlijke eindscore

Functioneel: 9/10

UX flow: 8.5/10

Visual polish: 7/10 → 10/10 met kleine tweaks

👉 Dit is geen “iets klopt niet”-app.
Dit is een “bijna af, nu afpoetsen”-app.
