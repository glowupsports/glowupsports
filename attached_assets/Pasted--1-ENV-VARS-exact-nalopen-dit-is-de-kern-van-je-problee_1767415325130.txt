A. Welke ENV VARS je minimaal nodig hebt

Voor Glow Up Sports (zoals jij ‘m bouwt):

🔑 Frontend (Expo / Mobile App)

Deze MOETEN beginnen met EXPO_PUBLIC_:

EXPO_PUBLIC_API_URL
EXPO_PUBLIC_SUPABASE_URL
EXPO_PUBLIC_SUPABASE_ANON_KEY
EXPO_PUBLIC_ENV=production


❗ Alles zonder EXPO_PUBLIC_ → bestaat NIET in Play Store

🔐 Backend (Replit / API)

Deze zijn server-only:

DATABASE_URL
SUPABASE_SERVICE_ROLE_KEY
JWT_SECRET
STRIPE_SECRET_KEY


Deze mogen NOOIT in de app komen.

B. Waar ze NU waarschijnlijk fout zitten
❌ Wat jij nu (waarschijnlijk) hebt

In Replit:

.env → werkt ✅

In Expo Preview:

preview.env → werkt half ⚠️

In Production (Play Store):

❌ ontbrekende env vars

❌ andere database URL

❌ undefined API

👉 Resultaat: functies doen niks / data “weg”.
