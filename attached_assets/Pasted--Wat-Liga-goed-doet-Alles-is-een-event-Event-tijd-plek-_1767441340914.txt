🔍 Wat Liga goed doet

Alles is een event

Event = tijd + plek + mensen + host

Geen “formulier”, alleen:

Sign up

Details

👉 De gebruiker denkt niet: “ik boek”
👉 De gebruiker denkt: “dit lijkt leuk, ik doe mee”

❌ Jouw huidige probleem

Je flow denkt nog als:

“Welke les wil je boeken?”

Dat is academisch, niet sociaal.

✅ Wat jij moet doen

Maak alles in Glow Up Sports een Session Card:

Session
- Type (Group / Semi / Private)
- Tijd
- Locatie
- Coach
- Spelers
- Status (open / full / waitlist)


❗️Geen losse “Book lesson” flow meer als startpunt

❌ Bij jou nu
