✨ 6. GLOW RULES (EXTREEM BELANGRIJK)

🧱 7. COMPONENT REGELS (HARD)
Buttons

Default: dark + stroke

Active: glow primary

Disabled: grey, geen glow

Cards

Default: bg.secondary

Selected: glow stroke

Important: subtiele inner glow (NOOIT fel)

Navigation

Inactive: text.muted

Active: glow primary + dot

Slechts 1 actieve kleur tegelijk

🧠 8. GAMIFICATION INTENSITY
State	Visual
Locked	Grey, flat
Unlocked	Color, no glow
Progressing	Color + bar
Level up	Color + burst glow (kort!)
Legendary	Pulse glow (tijdelijk)
