Ja. Dit is precies de juiste volgorde: eerst een 1-malige “Start Baseline” (intake / begin-assesment), daarna pas de “live engine” die meeloopt met lessen.

Hier is hoe we dat super uitgebreid, coach-proof, en 100% passend in jouw app/backend maken — zonder dat het een administratieve hel wordt.

1) Start Baseline (1-malig) — wat het moet zijn

Wanneer je op een speler klikt (Player Detail), moet je een knop hebben:

✅ “Set Starting Level (Baseline)”

1x verplicht bij nieuwe speler

daarna locked (alleen Admin/Owner kan unlocken, of met “override reason”)

Baseline bestaat uit 3 dingen:

Auto Level Suggestion (op basis van geboortedatum + ervaring)

Skill Baseline (super uitgebreid: FH/BH/Serve/etc + movement/mental/etc)

Trust/Confidence (hoe zeker zijn we van dit niveau)

2) Auto-level op basis van geboortedatum (default)

Dit gebeurt automatisch:

A) Leeftijd → “Recommended Track”

2–4 → Blue

4–8 → Red

8–10 → Orange

10–12 → Green

12–18 → Yellow (Youth)

18+ → Adult (UTR / League / Rating)

B) Binnen de track → start-level

De app kiest een startlevel met simpele intake vragen (30 sec):

“Hoe lang tennis je al?” (0–6m / 6–18m / 18m+)

“Speelt hij/zij wedstrijden?” (no / sometimes / often)

“Kan hij/zij 5 ballen over?” (no / yes)

“Serve lukt?” (no / basic / consistent)

➡️ Output: Suggested Level = Green 1 (bijv.) + confidence 0–100%

✅ Coach kan het 1x aanpassen → “Confirm Baseline”

3) Super uitgebreide baseline: maar tóch snel invulbaar

De truc: je maakt het diep in data, maar snappy in UX via 2 lagen:

Laag 1 — “Quick Baseline” (90 sec)

Coach ziet per pillar (6 pillars) een snelle keuze:

❌ Not yet / 🟡 Developing / ✅ Meets / ⭐ Above

Dit bepaalt meteen:

“overall readiness”

“what to focus next”

Laag 2 — “Deep Baseline” (5–12 min, alleen als nodig)

Hier ga je slag per slag en subskills doen.

En dit maak je modulair zodat het niet overwhelming is:

5) Ball-level bepaalt welke vragen je ziet (super belangrijk)

Je zei: “gebasseerd op backend wat we nodig hebben per ball lvl”

Dus:

✅ De app toont alleen relevante rubrics per ball level.

Voorbeeld:

Red: volley/overhead = “basic exposure” (minder diep)

Orange: volley/overhead = actief

Green: return + direction + patterns dieper

Yellow: match/tactics/serve/return volledig

➡️ Dat maakt het coach-proof én snel.

6) 1-malige manual adjustment (locked)

Je wil 1x aanpassen kunnen: top.

Flow:

Auto suggested level verschijnt

Coach drukt “Change”

Coach kiest nieuw level

Coach moet Reason selecteren (dropdown):

“Player clearly advanced”

“Late starter but athletic”

“Played in other academy”

“Competition experience”

“Age mismatch”

Coach → Confirm

Daarna:
🔒 Baseline locked

Alleen Admin/Owner kan “Unlock baseline”

Of coach kan “Request baseline change” (owner approves)

Dit voorkomt chaos.

7) Hoe dit in jouw Players lijst (screenshot) past

✅ Level Readiness % = “Hoe dicht bij volgende level”

als baseline ontbreekt: “Baseline needed” badge

Player row:

last baseline (Jan 6)

warning if no lessons
