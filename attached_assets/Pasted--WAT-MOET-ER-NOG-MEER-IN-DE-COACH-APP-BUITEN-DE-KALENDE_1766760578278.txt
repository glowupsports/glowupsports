🎾 WAT MOET ER NOG MEER IN DE COACH APP (BUITEN DE KALENDER)?
🧠 1️⃣ COACH DASHBOARD (LICHT & FUNCTIONEEL)

Doel: context geven, geen cijfershow.

Wat de coach ziet (bij openen app):

Vandaag:

Aantal lessen

Eerste les start over X min

Alerts:

Onbetaalde lessen vandaag

Spelers met holiday / afwezig

Snelle CTA:

“Ga naar kalender”

“Huidige les”

➡️ Geen grafieken, geen omzet, geen admin-info.

👥 2️⃣ PLAYER OVERVIEW (COACH-VIEW, MINI CRM)

Doel: coach kent zijn spelers zonder zoeken.

Player List

Naam

Ball level

Skill level

Laatste les datum

Status:

Holiday

Injured

New

Player Detail (tap)

Basisinfo

Medical notes

Attendance patroon (laat / afwezig)

Laatste coach focus

XP / progress (read-only)

➡️ Dit vervangt WhatsApp-geheugen.

🧾 3️⃣ SESSION HISTORY (READ-ONLY)

Doel: terugkijken zonder rommel.

Lijst van afgelopen lessen

Filter:

player

type

datum

Open session:

wie was aanwezig

feedback

intensity

notes

➡️ Geen edits, alleen context.

🧠 4️⃣ COACH MEMORY / NOTES HUB

Doel: coach onthoudt wat belangrijk is.

Korte notes per:

speler

groep

Auto-suggest in volgende les:

“Vorige focus: backhand consistency”

➡️ Slim, maar niet AI-zwaar.

🔔 5️⃣ NOTIFICATIONS CENTER (COACH-ONLY)

Doel: niets belangrijks missen.

Types:

Les begint over 10 min

Travel time waarschuwing

Week 9 recurring reminder

Holiday impact

Offline sync failed

➡️ Geen spam, alleen actiegericht.

🛠️ 6️⃣ COACH SETTINGS (BEWUST KLEIN)

Doel: persoonlijke flow optimaliseren.

Settings:

Default lesduur (60/90)

Default recurring (10 weken)

Default travel time

Focus Mode auto-on?

Notificatie voorkeuren

Offline sync gedrag

➡️ 1x instellen, daarna vergeten.

📶 7️⃣ OFFLINE STATUS & SYNC VIEW

Doel: vertrouwen op court.

Indicator:

Online / Offline

Pending sync items

Manual “Sync now” knop

Error retry

➡️ Geeft rust bij slecht internet.

🧑‍🏫 8️⃣ COACH PROFILE (LICHT)

Doel: identiteit, geen HR.

Naam

Foto

Locaties waar coach lesgeeft

Specialisaties (optioneel)

Availability (read-only)

➡️ Meer is niet nodig.

🚀 OPTIONELE / V2 MODULES (ALLEEN ALS JE WILT)
✨ 9️⃣ AI COACH ASSISTANT (LIGHT ENTRY)

“Wat was mijn drukste dag?”

“Welke speler komt vaak te laat?”

➡️ Geen chatty AI, vraag-antwoord only.

🎯 1️⃣0️⃣ COACH QUESTS / QUALITY CHECKS

Voor Glow DNA:

“3 lessen zonder late attendance”

“Geef feedback aan 5 spelers”

➡️ Motiveert kwaliteit, geen druk.

🚫 WAT WE NOG STEEDS NIET IN DE COACH APP DOEN

❌ Omzet / betalingen per maand

❌ Facturen

❌ Ouders berichten

❌ Admin tools

❌ Planning van andere coaches

Dat hoort elders.

🧠 SAMENVATTING (HEEL BELANGRIJK)

De Coach App bestaat uit 3 hoofdpijlers:

1️⃣ OPERATIONS

➡️ Kalender & sessies (hebben we volledig)

2️⃣ CONTEXT

➡️ Spelers, geheugen, history, alerts

3️⃣ FLOW & RUST

➡️ Focus Mode, offline, settings, notifications

👉 Als alle drie goed zijn → coach werkt beter, rustiger, consistenter.
