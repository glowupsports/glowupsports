🧬 De structuur (levels, XP, unlocks)

🔥 1. WAT IS HET SOLO LEVELING SYSTEM (IN JOUW APP)

Belangrijk:
Dit systeem meet NIET hoe goed iemand tennist
Het meet hoe “ver” iemand is in zijn journey in jouw app

Je hebt dus 2 volledig gescheiden assen:

A. Tennis Progress System (bestaat al)

Red / Orange / Green / Yellow

Glow Rank (skills + matches)

B. Solo Leveling (Account Level) ← DIT

Hoe actief

Hoe betrokken

Hoeveel gebruik van features

Hoeveel ownership over eigen progress

👉 Hierdoor:

Beginner speler ≠ domme app

Gevorderde speler ≠ overload

Iedereen groeit zichtbaar

🧬 2. SOLO LEVELING STRUCTUUR (OPEN & SCHAALBAAR)
🎮 Player Account Level (PAL)

Levels: 1 → 20 (initieel)

Waarom 20?
– Genoeg granulariteit
– Niet te grindy
– Later uitbreidbaar naar 30/50

🧱 LEVEL BLOKKEN (LOGISCH OPGEDEELD)
Level Range	Titel	Doel
1–3	Rookie	Begrijpen hoe app werkt
4–6	Player	Actief trainen & feedback
7–10	Competitor	Matches, analyse
11–15	Strategist	Data, patronen
16–20	Elite	Volledig ecosysteem

📌 Titels zijn puur UI / copy — makkelijk aanpasbaar

⭐ XP BRONNEN (GEEN TENNIS-SKILL!)

XP komt NOOIT uit “hoe goed je slaat”, maar uit gedrag:

Core XP Actions

Les bijwonen

Feedback ontvangen

Feedback lezen

Match spelen

Match evalueren

Profiel compleet maken

Consistentie (streaks)

Social (matches zoeken / accepteren)

📌 Dit maakt het cheat-proof
📌 Coaches hebben hier geen invloed op

🧮 XP IS GEWOGEN (BELANGRIJK)

Niet alles telt even zwaar:

Actie	XP	Type
Attend lesson	+10	Basic
Coach feedback received	+8	Basic
Feedback reviewed	+5	Engagement
Match played	+15	Competitive
Match evaluation completed	+10	Reflection
Weekly streak (3 sessions)	+20	Consistency
Profile completion	+15	One-time

👉 Replit kan deze weights aanpassen zonder systeemwijziging.

⏱️ 4. HOE LANG DUURT EEN LEVEL? (GEMIDDELD)

Dit is waar ik realistisch ben en Replit ruimte geef.

🎯 Design target (advies, niet vast)
Gebruikerstype	Gem. Level-up
Casual (1x/week)	3–4 weken
Actief (2–3x/week)	1,5–2 weken
Intensief (competitie)	5–10 dagen

📌 Belangrijk:

Eerste 3 levels = sneller (dopamine)

Levels 7+ = langzamer (waarde behouden)

📊 XP CURVE (NON-LINEAIR)

Level	XP nodig
