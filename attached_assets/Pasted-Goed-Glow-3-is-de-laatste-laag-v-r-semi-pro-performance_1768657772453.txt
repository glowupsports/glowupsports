Goed. Glow 3 is de laatste laag vóór “semi-pro / performance pathway”.
Hier maken we het verschil tussen:

“zeer goede competitieve speler”
vs
“speler die een traject verdient”

Glow 3 moet zwaar, eerlijk en coach-proof zijn.
Niet veel spelers mogen hier komen — en dat is precies waarom jouw systeem klopt.

🟣 GLOW 3 — ELITE / PERFORMANCE PATHWAY

(Van domineren → klaar voor performance-omgeving)

🧠 Filosofie Glow 3

De speler:

wint structureel

begrijpt waarom hij wint

kan zichzelf sturen

kan functioneren in een high-pressure omgeving

Glow 3 is geen eindpunt.
Het is een toelatingsexamen.

🟦 TECHNIQUE (PROOFED, NIET PERFECT)

Techniek hoeft niet mooi te zijn.
Het moet wedstrijd-proof zijn.

FOREHAND

⬜ FH is primaire identiteit (wapen)

⬜ Kan tempo + richting combineren

⬜ Verliest FH niet bij lange rally’s

⬜ Kan variëren: heavy / flat / short

⬜ FH blijft stabiel bij breakpoints

BACKHAND

⬜ BH is volledig betrouwbaar

⬜ Geen duidelijke “aanvalszwakte”

⬜ Kan druk absorberen én omzetten

⬜ BH DTL onder druk mogelijk

SERVE

⬜ Serve heeft duidelijke patronen

⬜ Tweede serve = veilig én assertief

⬜ Kan punten openen met serve

⬜ Double faults zeldzaam

RETURN

⬜ Leest serve-intenties

⬜ Kan return tactisch inzetten

⬜ Neemt initiatief wanneer nodig

NET / TRANSITION

⬜ Overgangen zijn bewust

⬜ Afmaken = efficiënt

⬜ Geen “hope volleys”

🟨 TACTICAL (MATCH IQ)

Glow 3 spelers denken vooruit.

⬜ Kan wedstrijdplan uitleggen

⬜ Past tactiek aan bij verlies

⬜ Snapt match flow & momentum

⬜ Speelt slim bij score (30-30, BP, TB)

⬜ Speelt tegenstander zwaktes consequent uit

🎯 Glow 3 = tactisch volwassen

🟩 PHYSICAL (COMPETITION READY)

⬜ Kan meerdere matches per week

⬜ Beweegt efficiënt (positioning > snelheid)

⬜ Herstelt tussen sets

⬜ Blessurebewust

⬜ Kan intensiteit verhogen wanneer nodig

🟦 MENTAL (STABIEL ONDER DRUK)

Hier vallen de meeste spelers af.

⬜ Emoties onder controle

⬜ Geen mentale instortingen

⬜ Kan verlies resetten

⬜ Blijft gefocust bij lange wedstrijden

⬜ Wil winnen, maar blijft rationeel

🧠 Glow 3 = mentale betrouwbaarheid

🟪 SOCIAL (PROFESSIONAL ATTITUDE)

⬜ Gedraagt zich als “serieuze speler”

⬜ Respecteert coach, tegenstander, regels

⬜ Kan feedback verwerken

⬜ Neemt verantwoordelijkheid

⬜ Kan functioneren in teamcompetitie

🟥 MATCH (STRUCTURELE RESULTATEN)

⬜ Wint >60–65% van competitieve matches

⬜ Kan winnen tegen gelijk/iets beter niveau

⬜ Verliest zelden van lagere niveaus

⬜ Resultaten over meerdere maanden

⬜ Match history vertelt een logisch verhaal

📊 Glow 3 = bewezen speler

🟣 PROMOTIECRITERIA → GLOW 4
HARD REQUIREMENTS

Minstens 8–12 officiële matches

Consistente performance (geen pieken/dalen)

Coach assessment = zeer positief

Geen structurele zwakte

WEGING

Match Performance: 30%

Mental: 25%

Tactical: 20%

Technique: 15%

Physical: 10%

👉 Glow 3 is match + mental driven.
