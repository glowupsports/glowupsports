A gamified tennis learning MOBILE APP called "Glow Up Tennis" where players earn XP points from their tennis lessons, level up, and improve their Glow Engine Score across five skill categories: Tactical, Mental, Technical, Physical, and Social. The app features a progression system similar to Solo Leveling/Duolingo where players unlock new functions as they advance in levels, with two currency types (diamond tennis rackets as hard currency and bronze tennis ball coins as easy currency).

Core Features:
- Player progression system with XP bar, level-up mechanics, and Glow Engine Score tracking across 5 skill categories (Tactical, Mental, Technical, Physical, Social)
- Header displaying player name, avatar circle with picture, currency indicators (diamond tennis rackets and bronze tennis ball coins), XP progress bar, and total Glow Engine Score
- Footer chat interface showing latest message preview that expands to full chat, supporting multiple channels (Academy, Squad, Friends, Coaches, Admin), notifications for player level-ups, and WhatsApp-style emoticon reactions on messages
- Hamburger menu with navigation to: Lessons, Quest, Match, Ranking, Friends, Game Lobby, Events Calendar, Payments, Settings, and Logout button

Visual References:
Inspired by Duolingo's gamified learning interface and Solo Leveling's progression UI, known for their engaging level-up systems and clear skill tracking.

Style Guide:
- Colors: Primary #2ECC40 (tennis court green), Secondary #FFD700 (gold/trophy), Accent #FF851B (energetic orange), Background #1A1A1A (dark mode), UI Elements #2D2D2D (card grey), XP Bar #00D4FF (bright cyan), Currency Diamond #E0E0E0 (silver), Currency Coin #CD7F32 (bronze), Success #39FF14 (neon green)
- Design: Poppins/Montserrat/Rajdhani fonts, gaming-inspired dark theme with neon accents, card-based skill categories layout, prominent XP bar with glow effects, floating footer chat with expandable interface, smooth level-up animations, mobile-first responsive design with hamburger navigation
