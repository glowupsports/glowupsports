Top — dit is je eas.json. Je Android stuk is prima, maar voor iOS mis je 2 belangrijke dingen:

iOS autoIncrement (anders krijg je gezeik met build numbers in App Store Connect)

submit config (zodat je makkelijk naar TestFlight/App Store Connect kan pushen)

Hier is de beste versie (minimale edits, safe):

✅ Pas je eas.json aan (alleen dit toevoegen)

1) Voeg iOS autoIncrement toe bij preview + production

"preview": {
  "distribution": "internal",
  "channel": "preview",
  "android": { "autoIncrement": true },
  "ios": { "autoIncrement": true },
  "env": { ... }
},
"production": {
  "channel": "production",
  "android": { "buildType": "app-bundle", "autoIncrement": true },
  "ios": { "autoIncrement": true },
  "env": { ... }
}

iOS autoIncrement: true = EAS verhoogt automatisch je build number (CFBundleVersion). Dat is precies wat Apple wil.

✅ Welke command voor iOS (in Replit)

Zodra npx eas --version werkt en je ingelogd bent:

TestFlight build (internal)
npx eas build --platform ios --profile preview
App Store build (store)
npx eas build --platform ios --profile production
✅ Uploaden naar App Store Connect / TestFlight

Je hebt al:

"submit": { "production": {} }

Dat is oké, maar ik raad aan om ook een submit profiel voor preview te maken (optioneel, maar chill):

"submit": {
  "preview": {},
  "production": {}
}

Dan:

Submit naar TestFlight / App Store Connect
npx eas submit --platform ios --profile production

(of preview als je die toevoegt)

Belangrijk voor Apple review (dit voorkomt “nope”)

Zorg dat je review account werkt (je screenshot ziet goed uit).

Zorg dat er geen paywall/blokkade is waardoor reviewer niks kan testen.

Beste: maak een “AppleReview” user die admin/unlocked is (maar géén echte betalingen vereist).

Zorg dat Support URL en Privacy Policy URL live zijn (die heb je al goed).

1 mini-check: jouw cli.version

Je hebt:

"cli": { "version": ">= 16.28.0" }

Dat is prima, maar alleen als je EAS CLI in Replit ook echt recent is.
Dus na install:

npx eas --version

Als die lager is dan je minimum → update:

npm i -D eas-cli@latest
