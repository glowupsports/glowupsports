✅ DEEL 1 — API ROUTES + PERMISSIONS

(REST-style, Supabase / Express / Next / Expo compatible)

Ik schrijf dit functioneel + security-proof, zodat:

coaches niks zien wat ze niet mogen

admins later kunnen overriden

spelers alleen hun eigen data raken

🔐 AUTH & ROLES (basis)
Rollen

coach

admin (later)

player (alleen holiday input)

Elke request heeft:

req.user = {
  id,
  role,
  location_ids[], // waar coach mag werken
}

🧑‍🏫 COACH API
📅 Calendar ophalen
GET /api/coach/calendar


Query

date=YYYY-MM-DD
view=day|week|month


Logic

Return:

eigen sessions (full data)

andere coaches → BLOCKED sessions (no details)

Alleen courts van coach locaties

Permission

role === coach

➕ Session aanmaken
POST /api/coach/sessions


Body

{
  court_id,
  location_id,
  start_time,
  duration,
  session_type,
  ball_level,
  skill_level,
  week_count,
  travel_time
}


Checks (server-side, verplicht)

coach beschikbaar?

court beschikbaar?

players (indien toegevoegd) beschikbaar?

travel_time check bij locatie wissel

Permission

coach mag alleen sessions maken op eigen locaties

✏️ Session aanpassen / verplaatsen
PATCH /api/coach/sessions/:id


Checks

ownership (coach_id === req.user.id)

conflict engine opnieuw uitvoeren

audit log schrijven

❌ Session cancelen
POST /api/coach/sessions/:id/cancel


Body

{ reason }


Logic

policy check (late cancel?)

status → cancelled

audit log

⏱️ Extend session
POST /api/coach/sessions/:id/extend


Body

{ minutes: 15 | 30 }


Logic

check court & coach availability

update end_time

👥 Spelers toevoegen / verwijderen
POST /api/coach/sessions/:id/players
DELETE /api/coach/sessions/:id/players/:playerId


Extra

guest player mogelijk (is_guest=true)

✅ Attendance opslaan (offline-safe)
POST /api/coach/sessions/:id/attendance


Body

{
  player_id,
  status,       // present/late/absent/holiday
  late_minutes,
  absence_reason
}


Logic

als offline → queue

bij sync → verwerken + XP trigger

🧠 Feedback & tags
POST /api/coach/sessions/:id/feedback


Body

{
  intensity, // light/normal/intense
  mood,      // good/neutral/low
  focus_tags,
  notes
}

📶 Offline queue sync
POST /api/coach/offline/sync


Logic

loop over offline_queue

replay actions

mark synced=true
